## Features

- Hacker News Firebase API

- Typescript, React, Next.js, Tailwind CSS

- SSR, alongside client-side comment tree fetching

## Development

```bash
git clone https://github.com/reiuyuu/nhn.git

cd nhn

pnpm

pnpm dev
```
